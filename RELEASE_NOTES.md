# MaxiTV 发布说明

---

## v1.1.0 (2026-06-09)

### ✨ 新功能

- **自定义频道添加** - 在频道页面点击 + 按钮，可单独添加自定义频道（独立于播放列表）
- **本地文件导入** - 设置中支持从本地导入 .m3u 播放列表文件
- **横屏/竖屏切换** - 播放界面新增横屏切换按钮，可自由切换屏幕方向
- **统一应用标题** - 所有 Tab 页面顶部居中显示 "MaxiTV" 标题

### 🎨 界面优化

- **App 图标** - 全新设计，支持所有屏幕密度（mdpi ~ xxxhdpi）
- **竖屏播放** - 视频完整显示，不裁剪
- **横屏播放** - 视频填满屏幕，沉浸式体验
- **频道页布局** - 减少顶部留白，标题更紧凑

### 🐛 问题修复

- 修复自定义频道播放时显示"频道不存在"的问题
- 修复竖屏播放时视频画面被裁剪的问题
- 修复包含中文字符的 URL 播放失败的问题
- 修复自定义频道添加后不显示在频道列表的问题

### 📦 技术改进

- 数据库版本升级至 v3，支持自定义频道表
- 新增 `CustomChannelEntity`、`CustomChannelDao`、`CustomChannelRepository`
- 播放器支持自定义 User-Agent 和 Referer 请求头
- URL 编码使用 `java.net.URI` 确保中文字符正确处理

## v1.0.0 (2026-06-08)

### 🎉 首个正式版本

### ✨ 核心功能

- **M3U 播放列表导入** - 支持从 URL 导入 M3U/M3U8 播放列表
- **频道分类与搜索** - 按分组浏览频道，支持关键词搜索
- **收藏频道** - 收藏喜爱的频道，快速访问
- **EPG 节目指南** - 显示电子节目单，查看当前和即将播放的节目
- **流畅播放** - 基于 Media3 ExoPlayer，支持 HLS、MPEG-TS 等流媒体协议
- **后台播放** - 支持后台音频播放
- **手势控制** - 左右滑动切换频道
- **主题切换** - 支持日间/夜间模式，可自适应跟随系统

### 🏗️ 技术架构

| 层级 | 技术 |
|------|------|
| 语言 | Kotlin 2.0 |
| UI | Jetpack Compose + Material 3 |
| 架构 | MVVM + Clean Architecture |
| 播放器 | AndroidX Media3 ExoPlayer 1.5 |
| 数据库 | Room 2.6 |
| 网络 | Retrofit + OkHttp |
| 依赖注入 | Hilt 2.50 |

### 📱 系统要求

- **最低版本**: Android 8.0 (API 26)
- **目标版本**: Android 15 (API 35)
- **推荐设备**: 手机、平板
